# Hero Carousel — 3D карусель категорий бассейнов

## Обзор

Замена текущей hero-секции (типографика + иконки категорий + кнопка "Рассчитать стоимость") на интерактивную 3D-карусель карточек с 10 категориями бассейнов. При выборе карточки фон секции плавно меняется на тематическое изображение.

## Подход

CSS 3D transforms (`rotateY` + `translateZ`) для полусферической траектории карточек. Чистый HTML/CSS/JS без внешних зависимостей.

## Структура слоёв (снизу вверх)

1. **Фоновый слой** — 10 полноэкранных изображений (по одному на категорию), стакнутых через `position: absolute`. Активный — `opacity: 1`, остальные — `opacity: 0`. Переход: `transition: opacity 0.8s ease`.
2. **Затемняющий оверлей** — полупрозрачный градиент для читаемости контента.
3. **3D-карусель** — контейнер с `perspective`. Внутри — вращающийся блок с 10 карточками, каждая повёрнута на `36° × i` и сдвинута по `translateZ()`.
4. **Кнопка "Перейти в раздел"** — под каруселью, привязана к активной категории. Ссылка-заглушка (`href="#"`).

## Карточка

- `border-radius: 20px`
- Фото бассейна на всю карточку (с Unsplash/Pexels)
- Подпись категории внизу (белый текст на затемнённом градиенте)
- **Активная:** `scale(1)`, яркая, тень
- **Соседние (±1):** `scale(0.8)`, затемнение 40%
- **Дальние (±2):** `scale(0.65)`, затемнение 60%
- **Остальные:** скрыты (за пределами видимости, `opacity: 0`)

## 10 категорий

1. Композитные чаши
2. Композитные бассейны
3. Строительство бассейнов
4. Сервисное обслуживание
5. Оборудование
6. Павильоны
7. Отделка бассейнов
8. Химия для бассейнов
9. Подогрев воды
10. Освещение

## Взаимодействие

- **Клик по карточке** — становится активной, карусель поворачивается к ней, фон crossfade-ом меняется
- **Свайп (мобильный)** — touch-события, переключение ±1 карточка
- **Бесконечный цикл** — после 10-й снова 1-я
- **Автопроигрывание** — нет, только ручное переключение
- **Стрелки** — нет

## Адаптивность

| Параметр | Desktop | Mobile (<768px) |
|---|---|---|
| Видимых карточек | 5 | 3 |
| Размер активной карточки | ~350×450px | ~250×320px |
| translateZ | ~450px | ~250px |
| Perspective | 1000px | 600px |

## Что заменяется

Полностью удаляется содержимое `.hero-inner`:
- Типографика "ХОРОШИЕ БАССЕЙНЫ" (`.hero-brand`)
- Полоска категорий-иконок (`.categories-wrap`)
- Кнопка "Рассчитать стоимость" (`.hero-cta`)
- Три столпа (`.hero-pillars`)

Навбар (`.nav-float`), top-bar, пузырьки (`.bubbles`) и волны (`.waves`) остаются без изменений.

Фон hero-секции заменяется на динамические изображения вместо текущих CSS-градиентов.
