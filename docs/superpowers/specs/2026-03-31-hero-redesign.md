# Hero Redesign — Good Pools

## Summary
Полная переделка index.html по прототипу: тёмный hero с крупной типографикой и горизонтальной каруселью категорий продукции. Замена текущего 644KB монолита чистым кодом.

## Header

**Расположение:** фиксированный, верх страницы.
**Фон:** светло-серый (#f5f5f5), высота ~60px.

| Позиция | Элемент |
|---------|---------|
| Лево | Гамбургер-иконка (≡) + текст "МЫ СТРОИМ" (caps, тёмный) |
| Право | 5 круглых иконок: чат, Telegram, WhatsApp, поиск, профиль — тёмные контурные, ~32px |

Гамбургер-меню пока без функционала (заготовка).

## Hero Section

**Размер:** 100vh, полноэкранный.
**Фон:** чёрный (#111111).
**Layout:** flexbox, два блока.

### Левая часть (~40%)
- "ХОРОШИЕ" — font-weight: 900, белый, ~90-110px, uppercase
- "БАССЕЙНЫ" — font-weight: 200-300, белый, ~90-110px, uppercase
- Текст выровнен по левому краю, вертикально по центру

### Правая часть (~60%) — Карусель категорий
Горизонтальный слайдер с 6 карточками. Видимы ~3.5 карточки одновременно (последняя обрезана, чтобы показать прокрутку).

**Карточка категории:**
- Ширина: ~220-250px
- Изображение продукта: PNG с прозрачным фоном, по центру, ~200px высота
- Название: две строки — жирное слово + тонкое слово (белый текст)
- Точка-индикатор: маленький серый круг под названием

**Навигация:**
- Стрелка вправо (>) на правом краю карусели
- Прокрутка по свайпу на мобильных
- Кликабельные точки-индикаторы

### 6 категорий

| # | Название | Картинка (временная) |
|---|---------|---------------------|
| 1 | КОМПОЗИТНЫЕ БАССЕЙНЫ | Вид сверху на прямоугольный бассейн |
| 2 | БАССЕЙНЫ КАСТОМ | Бассейн нестандартной формы |
| 3 | ГИДРОМАССАЖНЫЕ ДЖАКУЗИ СПА | Белая джакузи-ванна |
| 4 | НАДУВНЫЕ СПА | Круглое надувное спа |
| 5 | КУПЕЛЬ ФУРАКО | Деревянная купель-бочка |
| 6 | МЕБЕЛЬ ДЛЯ БАССЕЙНА | Шезлонг/мебель у бассейна |

**Изображения:** Временные из открытых источников. В будущем — загрузка через админ-панель. Формат PNG, прозрачный фон, ~400x400px.

## Технические решения

- **Шрифт:** Montserrat (Google Fonts) — уже используется
- **Иконки:** SVG inline или Font Awesome для соцсетей
- **Карусель:** чистый JS, CSS scroll-snap, без библиотек
- **Адаптив:** базовая мобильная адаптация (стек вертикально на <768px)
- **Деплой:** Vercel (уже настроен)
