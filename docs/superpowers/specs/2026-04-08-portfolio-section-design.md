# Секция «Портфолио» — Дизайн-спецификация

## Обзор

Секция портфолио размещается после секции «Почему выбирают нас» на главной странице. Показывает реализованные проекты в формате masonry-сетки с hover-эффектами и модальным окном деталей.

## Лейаут

- **CSS-only masonry** через `column-count: 3` (2 на мобилке)
- `column-gap: 20px`, `margin-bottom: 20px` на каждом элементе
- `break-inside: avoid` для предотвращения разрыва карточек
- `border-radius: 16px` на каждом элементе
- Максимальная ширина секции: 1440px, padding: 80px 60px (60px 20px на мобилке)

## Заголовок секции

Повторяет стиль секции «Каталог чаш»:

- **Лейбл:** «Наши работы» — 12px, uppercase, letter-spacing 5px, цвет #0ea5e9
- **Заголовок:** «ПОРТ» (bold 900, #e0f2fe) + «ФОЛИО» (thin 200, #0ea5e9), font-size 64px (42px мобилка)
- **Описание справа:** «Каждый проект — это история. От первого звонка до первого заплыва.» — 14px, #94a3b8
- **Кнопка:** «Все проекты →» — uppercase, 13px, font-weight 700, стрелка сдвигается вправо при hover

## Карточки проектов

Каждая карточка masonry-item содержит:

- **Фото** — `width: 100%`, разная высота (от 350px до 700px в зависимости от пропорций фото)
- **Hover-эффект:** фото `scale(1.05)` за 0.5s, появляется overlay с градиентом снизу вверх
- **Overlay:** `background: linear-gradient(to top, rgba(10,22,40,0.85) 0%, transparent 50%)`, opacity 0→1
- **Текст на overlay:** только название проекта, 18px, font-weight 700, белый

### Data-атрибуты на каждой карточке

- `data-name` — название проекта (напр. «Вилла в Сочи»)
- `data-model` — модель и размер (напр. «HIIT • 8×3.6м»)
- `data-desc` — краткое описание проекта (1-2 предложения)

## Модальное окно

При клике на карточку открывается модалка:

- **Backdrop:** фиксированный, rgba(10,22,40,0.9), backdrop-filter blur(8px)
- **Модалка:** background #0f2035, border-radius 24px, max-width 800px, width 90%
- **Кнопка закрытия:** ×, абсолютная позиция top-right, круглая 36px, rgba(0,0,0,0.5)
- **Фото:** width 100%, max-height 500px, object-fit cover
- **Инфо-блок (padding 28px):**
  - Название: 24px, font-weight 800, #e0f2fe
  - Модель: 12px, font-weight 600, #0ea5e9, uppercase, letter-spacing 2px
  - Описание: 14px, #94a3b8, line-height 1.7
- **Закрытие:** клик по backdrop или кнопке ×

## CTA-блок внизу

- margin-top 50px, padding 40px, text-align center
- background: linear-gradient с rgba(14,165,233,0.08)→rgba(6,182,212,0.12)
- border-radius 24px, border 1px solid rgba(14,165,233,0.15)
- **Заголовок:** «Хотите такой же?» — 18px, #e0f2fe, font-weight 600
- **Подзаголовок:** «Покажем ещё проекты и подберём решение под ваш участок» — 14px, #94a3b8
- **Кнопка:** «Обсудить проект» — gradient #0ea5e9→#0284c7, padding 16px 40px, border-radius 14px
